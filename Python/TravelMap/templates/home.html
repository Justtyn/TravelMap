Smart Travel Backend

一站式旅游内容与交易引擎

TravelMap 是一个基于 Flask + SQLite 的文旅应用后端，实现用户认证、景点 / 商品管理、 收藏、行程计划、购物车与订单全链路能力，适合课程设计、原型验证和轻量部署。

查看 GitHub 仓库 ↗ 阅读文档

技术栈亮点

  • Flask 3 + 原生 sqlite3，零依赖即可部署。
  • RESTful JSON 接口，统一响应规范。
  • tests_smoke.py 自带多模块调用示例。

项目能力

  • 用户体系：本地账号 + 微信占位登录。
  • 内容侧：景点、商品、预订接口全量返回。
  • 交易链路：收藏、去过、行程、购物车与订单。

部署建议

  • 服务器执行 python app.py 即可启动 API + 前端。
  • 数据库 db/TravelMap.db 已注入真实感样本。
  • 可直接通过 Flask 提供的静态页面访问本站。
